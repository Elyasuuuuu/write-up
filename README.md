Each write-up follows a consistent layout:

1. **Introduction** — room overview and objectives  
2. **Reconnaissance / Enumeration** — how the attack surface was mapped  
3. **Exploitation** — vulnerability identification and abuse  
4. **Post-exploitation** — privilege escalation and flag retrieval (without revealing flag values)  
5. **Conclusion** — key takeaways
